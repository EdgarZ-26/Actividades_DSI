Tarjetas de tickets

Ticket #001

Alta

Error crítico en el sistema

El sistema no permite el acceso a los usuarios registrados. Se requiere atención inmediata para restablecer el servicio.

Categoría: Sistema

Estado: Pendiente

Ticket #002

Media

Actualización de información

Se solicita modificar los datos de contacto de un usuario dentro de la plataforma administrativa.

Categoría: Gestión de usuarios

Estado: En revisión

Ticket #003

Baja

Consulta general

El usuario solicita información sobre el funcionamiento básico de una sección del sistema.

Categoría: Soporte técnico

Estado: Resuelto
